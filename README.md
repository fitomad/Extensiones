####UIColor (HEXtension) ####
Crea un _UIColor_ a partir de una representación hexadecimal como la que se usa en HTML.

```Objective-C
UIColor *color = [UIColor colorWithHexString:@"#f7f7f7"];
```
o también...
```Objective-C
UIColor *color = [[UIColor alloc] initWithHexString:@"#f7f7f7"];
```
